Print Widget

This widget is designed to print simple feature reports using markdown configuration. The markdown configured in the settings can include fields from the datasource, wrapped in ${}. You can also perform some simple formatting on date or number fields, using the following syntax:
Date: ${dateField | "M/D/YY h:mm A"} => 2/15/26 4:23 PM
Number: ${numField | "#,###.00"} => 23,456.78

The formatting string uses Arcade syntax, similar to the syntax in the Text() Arcade function
